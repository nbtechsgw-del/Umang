# Smart EMI Calculator

## 📌 Project Description
A full-stack EMI Calculator that calculates monthly loan EMI using React (frontend) and Spring Boot (backend).

## 🚀 Features
- Calculate EMI instantly
- Backend API integration
- Input validation
- Clean UI
- Displays total payment and interest

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Spring Boot (Java)
- API: REST

## ▶️ How to Run

### Backend
cd backend
mvn spring-boot:run

### Frontend
cd frontend/emi-calculator
npm install
npm start
